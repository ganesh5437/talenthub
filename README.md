# TalentHub — AI-Powered Recruitment & ATS Platform

Full-stack Applicant Tracking System with 4 role-based portals: SuperAdmin, Company Admin, Recruiter, Candidate.

## Features
- Complete ATS pipeline: job posting → application → screening → interview → offer
- Interview scheduling system
- Real-time notification bell (polling-based)
- Role-based access control via JWT
- Company & recruiter management

## Tech Stack
**Frontend:** React, Vite, Tailwind CSS
**Backend:** Node.js, Express.js
**Database:** PostgreSQL (Neon) via Prisma ORM
**Auth:** JWT
**Deployment:** Backend on Render, Frontend on Netlify
